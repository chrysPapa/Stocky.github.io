Our Philosophy

This project aims to create a website that will be used by stock traders to find relevant data and news by collecting datasets, grouping them, and analysing them based on keywords. //Then compare them and produce a general analytic panel that then will be present back to the user. Some data that will be presented will be (article counter, Positive key words counter and percentage, Negative Keywords counter percentage).

Some of the issues that we want to address

How an Individual can be surer that what he/she is reading is true or fake
.How to identify fake news
.How to label news as true
.How an individual can be consistent in his/her decisions
.Is there a way to answer all the above questions

How we are adressing the issues

A solution to the problem would be to compare massive data collections of sources and articles. By doing so slowly we start to filter out fake news by counting the repetition of the data.

Example

Let’s say that we have 100 articles focused on the same topic we start by searching for all the similarities filter them out and then rate them based on how many out of the 100 had similar data. By doing so we can say that the data is relevant us they are back up by many sources and can be more trusted. Furthermore, by analysing the context of the articles we can also judge if the articles have positive/natural/negative slang that will also be used to create a percentage margin. So now the user will have access to the following. That can be used to back up his own research and have an extra perspective.
. A list with similar data
. A percentage of how positive/natural/negative the articles are.
. A list of the negative articles if they want to feather search why some articles are negative
